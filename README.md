# OtterScrap

A Node.js + TypeScript project using Puppeteer to scrape web pages.

## Setup

1. Install dependencies:
   ```sh
   npm install
   ```

2. Build the project:
   ```sh
   npm run build
   ```

3. Run the scraper (after build):
   ```sh
   npm start
   ```

## Usage

Edit `src/index.ts` to change the scraping logic or target URL.
